== Description ==

= 사용 할 수록 홈페이지가 자연적으로 홍보되는 차세대 소셜댓글 서비스 입니다. =

* [홈페이지](https://www.cosmosfarm.com)
* [사용자 커뮤니티](https://www.cosmosfarm.com/threads)
* [설치방법](https://blog.naver.com/PostView.nhn?blogId=chan2rrj&logNo=220395268358)

워드프레스 페이지, 포스트, KBoard(케이보드) 게시판 등 다양한 곳에 소셜댓글을 추가할 수 있습니다.

**쉬운 설치 방법**

전문가가 아니어도 괜찮습니다. 단 몇 줄의 코드를 추가하거나 워드프레스 플러그인을 설치하는 것으로 당신의 웹 사이트에 코스모스팜 소셜댓글을 설치할 수 있습니다. 즉시 더 많은 사용자들의 이야기를 들을 수 있습니다.

**모두가 참여 가능한 커뮤니티**

내 사이트의 회원과 소셜회원 모두 참여가 가능한 커뮤니티를 구축할 수 있습니다. 네이버, 카카오톡, 페이스북, 트위터, 구글 등 다양한 소셜로그인을 지원하며 내 사이트 회원까지 토론에 참여할 수 있어 더 많은 사용자를 확보할 수 있습니다.

**강력한 리얼타임 채팅**

실시간으로 댓글이 달려 페이지에 함께 있는 사용자끼리 채팅이 가능합니다. 화면을 보고 있지 않아도 단 하나의 대화도 놓치지 않도록 즉시 이메일로 알려드립니다. 사용자간 커뮤니티를 구축하기 위해서는 반드시 필요한 기능 입니다.

**편리한 부가 기능**

유튜브 주소만 입력하면 동영상 플레이어가 자동 생성되어 모바일 환경에서도 쉽게 동영상 공유가 가능합니다. 커뮤니티에 필요한 각종 편의 기능들이 지속적으로 추가됩니다.

**꼭 필요한 관리자 기능**

통합 댓글관리, 엑셀파일 다운로드, 관리자 알림, 비밀글, 로그인 API, 링크주소 차단, 스팸관리, 디자인 변경등 다양한 기능들을 손쉽게 관리할 수 있습니다.

**사이트 방문자 상승**

소셜네트워크의 시대에서도 구글, 네이버등 검색엔진에서 가장 많은 방문자가 발생합니다. 코스모스팜의 소셜댓글에서 작성된 댓글은 검색엔진의 검색결과에서 소셜댓글이 설치된 페이지로 연결시켜 사이트 방문자가 늘어납다.
