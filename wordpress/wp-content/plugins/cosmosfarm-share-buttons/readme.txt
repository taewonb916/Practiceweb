== Description ==

= 소셜 공유 버튼 By 코스모스팜은 국내외 SNS 소셜 공유하기 버튼을 제공합니다. =

* [무료 게시판 KBoard 플러그인 다운로드](https://www.cosmosfarm.com/products/kboard)
* [홈페이지](https://www.cosmosfarm.com)
* [사용자 커뮤니티](https://www.cosmosfarm.com/threads)

워드프레스 페이지, 포스트, 요약글, KBoard(케이보드) 게시판, 포트폴리오, 우커머스 상품 상세페이지 등 삽입을 지원합니다.

네이버, 페이스북, 트위터, 밴드, 카카오스토리, 카카오톡, 구글 플러스, 라인 등 국내외 SNS 소셜공유 버튼 기능을 제공합니다.

네이버 오픈메인 기능이 추가되어 여러분의 사이트를 네이버 메인에 쉽게 추가할 수 있게 되었습니다.

숏코드를 사용해서 원하는 위치에 소셜 공유 버튼을 삽입할 수 있습니다.
