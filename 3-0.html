GPA 계산기 공통교육 과목 졸업요건 확인

GPA 계산기 사용법 보기 ▽

  1. 각 학점의 단위 수를 입력합니다.
  2. 계산을 누릅니다.

공통교육 과목 사용법 보기 ▽

  1. 학부, 학과, 입학년도를 선택하고 확인을 누릅니다.
  2. pdf를 열어 학과 별 공통교육 과목의 졸업요건 단위 수를 확인합니다.
  3. 각 과목에 단위 수를 입력 후 계산을 누릅니다.
  4. 선택한 학과의 공통교육 과목의 졸업요건 단위 수와 부족한 단위 수를 확인합니다.

졸업요건 확인 사용법 보기 ▽

  1. 학부, 학과, 입학년도를 선택하고 확인을 누릅니다.
  2. pdf를 열어 학과 별 졸업요건 단위 수를 확인합니다.
  3. 각 과목에 단위 수를 입력 후 계산을 누릅니다.
  4. 선택한 학과의 졸업요건 단위 수와 부족한 단위 수를 확인합니다.
